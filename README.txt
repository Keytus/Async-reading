Вариант 1:
1 Обработка содержимого файла, используя асинхронные операции
Задан файл Test.txt, содержащий набор произвольный латинских букв. Постановка задачи: считать определенное количество символов из файла и изменить регистры символов. Считывание может проводится традиционно и асинхронно. Для асинхронного можно задавать количество инициализируемых задач. Выводится время за сколько отработал какой способ.
